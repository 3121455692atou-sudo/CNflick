## 核心特性
- 12 键 Flick 输入
- 三语言切换：拼音 / 注音 / 日语假名
- 地球键可配置：
  - 可作为语言切换键
  - 可限制参与切换的语言集合（例如仅假名 + 注音）
  - 可隐藏或禁用
- 多层候选与学习：
  - 中文支持分段候选与整句候选
  - 日语支持分段候选与整串候选
- 词库体系：
  - 中文词库（拼音与注音共享）
  - 日语词库（独立）
  - 快捷词库（自定义输入码到短语）
- 映射自定义：
  - 拼音 / 注音 / 日语 / 符号映射分别管理
  - 支持 JSON 导入导出
  - 支持八方向滑动映射（按类型独立开关）
- 主题系统：
  - 主题色、字体、背景图、按键图、按键音效
  - 一键导入主题包
  - 预设主题包 + 已导入主题包可切换

## 快速上手
1. 在系统输入法设置里启用 `CNflick`。
2. 切换到 `CNflick`。
3. 左下角地球键可在已启用语言间轮换。
4. 进入设置可完成：语言集合配置、词库配置、映射调整、主题与外观定制。
