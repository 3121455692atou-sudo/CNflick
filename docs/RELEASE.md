# CNflick 发布说明

## v1.2
- 移除系统语音输入功能（含相关开关和权限）。
- 八方向滑动输入拆分为独立开关：
  - 拼音八方向默认关闭。
  - 符号八方向默认开启。
- 新增按键文字显示独立开关：
  - 显示/隐藏中间大字。
  - 显示/隐藏四周小字。
- 八方向斜向映射会在按键四角同步显示小字。
- 符号默认映射重构为 12 键 * 9 向，括号成对、方向统一。
- 自定义映射保持二级/三级结构：正向在二级，斜向在三级。

APK：
- `release/CNflick-v1.2-debug.apk`
- `release/CNflick-v1.1-debug.apk`（保留）
- `release/CNflick-debug.apk`（保留）
