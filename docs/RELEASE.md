# CNflick 发布说明

## v1.3
- 新增输入语言体系：
  - 拼音 / 注音 / 日语假名可在同一输入法中切换。
  - 地球键支持语言切换与可循环语言集合配置。
- 新增注音输入模式：
  - 输入框显示注音。
  - 后台转换拼音并复用中文词库。
  - 注音与拼音映射独立可配。
- 新增日语输入模式：
  - 日语假名映射独立可配。
  - 日语词库与中文词库分离管理。
  - 新增日语变换键 `゛゜小`（默认仅点按，不带方向输出）。
- 优化日语输入流：
  - 支持分段候选与整串候选。
  - 未选候选直接回车可上屏假名原串。
- 优化注音键位：
  - 调整为 1~9/*/0/# 终版布局。
  - 兼容将 `ㄒ` 放在介音区的声母处理。
- 主题包导入容错增强：
  - 支持 `theme-pack.json` 常见别名与大小写差异。
  - 支持清单/主题 JSON 位于子目录。
  - 资源路径支持更宽松匹配与常见默认名推断。
- 日语内置词库扩容：
  - 扩展为大规模读音词条，常用词命中显著提升（如 `あい -> 愛`）。

## v1.2
- 相对 v1.1 新增八方向滑动输入能力，并拆分为独立开关：
  - 拼音八方向默认关闭。
  - 符号八方向默认开启。
- 相对 v1.1 新增按键文字显示独立开关：
  - 显示/隐藏中间大字。
  - 显示/隐藏四周小字。
- 相对 v1.1 新增斜向映射显示：斜向字符会同步显示在按键四角小字。
- 相对 v1.1 新增映射菜单分层：
  - 二级菜单配置正向映射。
  - 三级菜单配置斜向映射。
- 相对 v1.1 重构符号默认映射为 12 键 * 9 向，括号成对、方向统一。
- 相对 v1.1 新增主题包能力：
  - 一键导入主题包（主题/字体/背景图/按键图/音效）。
  - 一键切换已导入主题包。
  - 预设主题包：纯默认主题包、初音主题包（含 Mono 字体）。
  - 新增主题包模板示例：`release/theme-packs/cnflick-theme-pack-template-pink/`。

## v1.1
- 外观系统：背景图/按键图导入后自动裁切并进入可选列表。
- 新增背景与按键图推荐比例提示（背景 1:1，按键 2.45:1）。
- 默认背景可选纯色与初音背景图（移除默认初音按键图）。
- 词库管理页：支持导入词库、启用多个词库。
- 新增快捷词库：支持自定义码表映射（如 `js -> 小学生`）。
- 引入全量雾凇主词库与开源动漫中文词库（默认启用）。
- 候选引擎优化：长拼音支持分段选词与整句学习，减少无意义乱候选。
- 性能优化：候选查询异步化与缓存优化，减少卡顿/无响应。
- 新增输入法收起按钮。
- 主题/字体/外观/音效均支持恢复默认。
- 修复设置页闪退问题。

  - 英文/数字/符号/功能副键盘
  - 剪贴板历史
  - 按键音/按键震动开关
  - 自定义按键映射 + JSON 导入导出
  - 底部快捷区：地球切换输入法 / 设置入口
