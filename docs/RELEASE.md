# CNflick 发布说明

## v1.1
APK：
- 1.0（保留）：`release/CNflick-debug.apk`
- 1.1（新增）：`release/CNflick-v1.1-debug.apk`
- GitHub Release：
  - https://github.com/3121455692atou-sudo/CNflick/releases/tag/v1.1.0

相较 v0.1.0 / 1.0 的主要新增与修复：
- 新增外观系统：背景图/按键图导入后自动裁切并进入可选列表
- 新增背景与按键图推荐比例提示（背景 1:1，按键 2.45:1）
- 默认背景可选纯色与初音背景图（移除默认初音按键图）
- 新增词库管理页：支持导入词库、启用多个词库
- 新增快捷词库：支持自定义码表映射（如 `js -> 小学生`）
- 引入全量雾凇主词库与开源动漫中文词库（默认启用）
- 候选引擎优化：长拼音支持分段选词与整句学习，减少无意义乱候选
- 性能优化：候选查询异步化与缓存优化，减少卡顿/无响应
- 新增输入法收起按钮
- 主题/字体/外观/音效均支持恢复默认
- 修复设置页闪退问题
