# CNflick 使用说明（v1.3）

## 1. 启用输入法
1. 打开 CNflick。
2. 点击“启用输入法”，在系统输入法列表中开启 `CNflick`。
3. 点击“切换输入法”，选择 `CNflick`。

## 2. 输入模式
CNflick 采用 12 键 Flick，支持三种语言：
- 拼音
- 注音（显示注音，后台转拼音查中文词库）
- 日语假名

左下角地球键默认用于“切换输入语言”。

## 3. 地球键与语言切换
入口：设置 -> 外观设置

可配置项：
- 地球键切换语言：开/关
- 地球键可循环语言：
  - 可单独启用/禁用拼音
  - 可单独启用/禁用注音
  - 可单独启用/禁用日语假名
- 地球键显示模式：
  - 正常显示
  - 隐藏
  - 显示但禁用

说明：
- 至少会保留一种可用输入语言。

## 4. 中文输入（拼音 / 注音）

### 4.1 拼音
- 按“声母 -> 韵母”输入。
- 候选栏会给出词和短句。
- 空格键在有候选时优先上屏首候选。

### 4.2 注音
- 输入区显示注音。
- 后台自动转换为拼音，使用同一套中文词库。
- 回车或空格时，如未选汉字候选，会直接上屏当前注音串。

### 4.3 中文长句
- 支持分段选词。
- 支持整句候选优先。
- 选词会参与学习，后续排序会更贴合你的习惯。

## 5. 日语假名输入

### 5.1 基本输入
- 采用 12 键日语 flick 布局。
- 输入区显示假名，候选区给出汉字/词语。

### 5.2 变换键（゛゜小）
- 变换键为“点按键”，默认不提供滑动方向输出。
- 用于变换最后一个假名：
  - 清音/浊音（例：か <-> が）
  - 半浊音（例：は -> ば -> ぱ）
  - 小假名（例：つ -> っ）

### 5.3 日语句子输入
- 支持整串候选。
- 支持前缀分段候选（可边选边输入）。
- 若不选候选直接回车，会直接上屏当前假名串。

## 6. 候选与上屏规则
- 点击候选：上屏候选文字。
- 空格键：
  - 有候选时上屏第一候选。
  - 无候选时上屏原始输入串或空格。
- 回车键：
  - 有未上屏输入时，优先上屏原始输入串。
  - 无输入时发送回车事件。

## 7. 词库管理
入口：设置 -> 词库管理（导入 / 多词库启用）

词库分层：
- 中文词库页：拼音与注音共用
- 日语词库页：假名单独

支持：
- 启用/禁用多词库
- 导入 JSON/TXT/TSV
- 恢复默认开关

快捷词库（中文页）：
- 可设置如 `js -> 小学生`
- 可新增、删除

## 8. 映射自定义
入口：设置首页

可分别配置：
- 自定义拼音映射
- 自定义注音映射
- 自定义日语假名映射
- 自定义符号映射

说明：
- 映射支持 JSON 导入导出。
- 映射编辑为分级菜单：
  - 二级：中/左/上/右/下
  - 三级：左上/右上/左下/右下（需启用八方向）

## 9. 外观与主题
入口：设置 -> 主题设置 / 字体设置 / 外观设置 / 按键音效设置

可配置：
- 主题色
- 主题包（含字体/背景/按键图/音效）
- 字体
- 输入法背景图
- 按键图片
- 按键音效
- 透明度、字号、按键间距、按键尺寸
