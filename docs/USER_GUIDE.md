# CNflick 使用教程（v1.1）

## 1. 启用输入法
1. 打开 App 首页。
2. 点击“启用输入法”，在系统输入法列表中开启 `CNflick`。
3. 点击“切换输入法”，选择 `CNflick`。

## 2. 基础输入
CNflick 采用 12 键 Flick：
- 拼音区：先输入声母，再输入韵母。
- 候选区：点击候选上屏。
- 空格键：有候选时优先上屏第一个候选。

## 3. 长拼音输入（重点）
例如输入 `babawolaile`：
- 候选会优先给常见长句或可分段短词。
- 你可以先选前半句，再选后半句，最终完成整句。
- 本次整句结果会被学习，下次可直接更快命中。

## 4. 词库与快捷词库
入口：设置 -> `词库管理（导入 / 多词库启用）`

可用功能：
- 多词库同时启用/禁用
- 导入自定义词库（JSON/TXT/TSV）
- 快捷词库（例如：`js -> 小学生`）
- 快捷词条可随时添加与删除



## 5. 外观与主题
入口：设置 -> `外观设置`

### 背景与按键图
- 输入法背景推荐比例：`1:1`（导入后自动裁切）
- 按键图片推荐比例：`2.45:1`（导入后自动裁切）
- 每次导入会自动加入可选列表

### 可调参数
- 中心/四周文字大小
- 按键文字透明度
- 按键图片透明度
- 按键底色透明度（可调到 0）
- 按键整体大小
- 按键间距

## 6. 主题包
入口：设置 -> `主题设置`
- 可导入 `.zip` 主题包
- 支持恢复默认主题

制作教程见：`docs/THEME_PACK.md`

## 7. 字体
入口：设置 -> `字体设置`
- 导入 ttf/otf 后可切换
- 字体会全局应用到输入法界面
